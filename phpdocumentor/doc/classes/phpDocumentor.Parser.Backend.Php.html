parse()

parse(\SplFileObject  $file) : void

Analyzes the file's contents as a PHP file and stores the result in the Project Descriptor in the Analyzer that is associated with this object.

When an analyzed file with the exact same path is present in the Project Descriptor and its contents do not differ from the provided file (calculated by hash) than the analysis is skipped and the existing file in the Project Descriptor is considered valid.

This method also dispatches the \EVENT_FILE_IS_CACHED event if the file is cached and not re-analyzed.

Parameters

\SplFileObject $file

analyzeFile()

analyzeFile(\SplFileObject  $file) : void

Analyze the given file and store the results in the Project Descriptor.

Please note that the persisting of a FileDescriptor is part of the analyze() method of the Analyzer for PHP files. That means that the code in this method does not show that the File Descriptor is persisted but it is.

This method also dispatches the \EVENT_ANALYZED_FILE event.
