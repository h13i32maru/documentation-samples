resolvePropertyParents(node) → {Array.<module:jsdoc/doclet.Doclet>}

Given an AST node representing an object property, find the doclets for the parent object or objects. If the object is part of a simple assignment (for example, `var foo = { x: 1 }`), this method returns a single doclet (in this case, the doclet for `foo`). If the object is part of a chained assignment (for example, `var foo = exports.FOO = { x: 1 }`, this method returns multiple doclets (in this case, the doclets for `foo` and `exports.FOO`).
