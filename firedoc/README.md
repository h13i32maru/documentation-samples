# 特徴
## ES対応
- シンタックスを参照していないので問題なさそう

## YUI fork
- YUIをforkしている
- なので大体YUIと同じぽい

## i18n
- `` -L --lang <language>  the i18n language``
- 国際化対応できるけど、いまのところen, zhのみ

```javascript
/**
 * !#en
 * English class description
 * !#zh
 * 中文类型描述
 * @class FLogic
 */
```

## 出力
- HTML, Markdownに出力できる
- Markdown: https://github.com/fireball-x/firedoc/tree/master/docs
